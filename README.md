## Quick start
- Clone github repository
```
git clone https://github.com/Sosokker/Food-Nutrient-Viewer-Tkinter
```
- run setup.py or pip
```py
# pip
pip install -r requirements.txt
```
```py
# or setup.py
python setup.py install

```

- To start GUI window run app.py
```py
python app.py
```
